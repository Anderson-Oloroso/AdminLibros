# Gestionador de libros, músicas y películas
### Nombre del repositorio 🚀: _AdminLibrosMusicasPelis_ 

#### ✅ Descripcion del proyecto:
Aplicacion de consola desarrollado en Python para gestionar los inventarios de peliculas, musicas y libros, este programa es capaz de :
- Agregar 
- Listar 
- Actualizar 
- Eliminar
-- > Tanto a los libros como a las musicas y las peliculas 

## 👾 *Como ejecutar el proyecto*
1. Crear un directorio especifico para el proyecto
2. Clonar el repositorio en el direcotrio especificado:
```bash
git clone https://github.com/Anderson-Oloroso/AdminLibrosMusicasPelis.git
```
3. Abrir el proyecto
4. Ejecutar el archivo "main.py"
